# Komentář k zákonu č. 117/1995 Sb., o státní sociální podpoře (SSP)

Odborný komentář k **zákonu o státní sociální podpoře** — předpisu upravujícímu nepojistné rodinné dávky. Komentář pokrývá **všech 71 paragrafových jednotek** zachované normativní úpravy (§ 1–81 s mezerami).

- **Konsolidované znění:** k **1. lednu 2026** (e-Sbírka, dokument `SB-1995-00117_2026-01-01`, document_id 2055763).
- **Rozsah:** ~33 tis. slov, 5 obsahových kapitol + úvod + index.
- **Tři dávky:** rodičovský příspěvek, porodné, pohřebné. Pojmy rodina, nezaopatřené dítě, rozhodný příjem; organizace a řízení (Úřad práce ČR, MPSV).

## Metoda

Komentář je zpracován **výhradně z lokálních materiálů** — z úplného konsolidovaného znění zákona. U každého paragrafu je uvedeno **úplné doslovné znění** (bez zkracování elipsami) a poté **odborný výklad** členěný do tematických podnadpisů, včetně vztahu k souvisejícím předpisům (zákon o životním a existenčním minimu, správní řád) a k právu EU (koordinace rodinných dávek). **Konkrétní spisové značky judikatury nejsou vymýšleny.**

Zákon vynechává rozsáhlé bloky paragrafů (zrušené dávky); tyto anomálie jsou věrně zaznamenány.
